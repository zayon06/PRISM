# PRISM

short for **Platform for Real-time Intelligent Skills Mapping.**
This is a SaaS platform designed to eliminate the Talent Mismatch by transforming static HR records into a dynamic, searchable, and intelligent internal talent marketplace

# client

move to the client directory

```bash
cd client
```

start nextjs dev server

```bash
pnpm dev
```

# services

move to a service dir

```bash
cd services/<service-name>
```
